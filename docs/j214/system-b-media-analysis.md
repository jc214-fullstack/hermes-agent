# System B Media Analysis Pipeline

System B is Mike/J214's Hermes media-analysis pipeline for Discord intake threads. It processes mixed media sources — YouTube, Instagram, social posts, documents, raw uploads, images, video, and articles — into durable thread workspaces and a source index.

The old `instagram-reel-analyzer` name was a prototype label and is no longer the canonical identity. The canonical name is **System B Media Analysis Pipeline** / **Hermes Media Analysis Pipeline**.
